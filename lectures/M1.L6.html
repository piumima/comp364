The Human Compendium (HuC)

  • Throughout this course, we will examine breast cancer -omic datasets.
  • We've made available a compendium of some available gene expression datasets.
  • It is organized as an R data.frame.
  • First we load a file which contains a set of functions that we wrote to load and process the HuC into R.
